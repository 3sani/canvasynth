Class: Oscillator

Oscillator(typeopt, frequencyopt, amplitudeopt, xoffsetopt, yoffsetopt)

An oscillator that produces the values of a standard wave.

Constructor

new Oscillator(typeopt, frequencyopt, amplitudeopt, xoffsetopt, yoffsetopt)

Parameters:
Name Type Attributes Default Description
type number <optional>
 0 Type of wave. 0 = sine, 1 = square, 2 = triangle, 3 = sawtooth.
frequency number <optional>
 1 Number of wave periods per synthesisation period.
amplitude number <optional>
 1 A number the wave is multiplied by. All waves are trunctated to (-1,1)
xoffset number <optional>
 0 Amount of x-offset as a fraction of synthesisation period.
yoffset number <optional>
 0 Amount of y-offset as a total value (-1, 1).
