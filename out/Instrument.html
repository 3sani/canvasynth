Class: Instrument

Instrument()

An abstract "visual instrument". Draws shapes etc.

Constructor

new Instrument()

Properties:
Name Type Description
name String Name of the instrument. Used for UI.
color String Color in a format that HTML canvas accepts, such as "rgb(255, 255, 255)".
xOsc Object Oscillator that defines the instrument's render position on the x-axis.
yOsc Object Oscillator that defines the instrument's render position on the y-axis.
alphaOsc Object Oscillator that defines the instrument's opacity. Can also be set to "constant" to have the oscillator be constantly opaque.
enabled boolean Whether or not this instrument is enabled and should be drawn.
maxWidth Number Maximum width the instrument's output can be.
maxHeight Number Maximum ehight the instrument's output can be.
